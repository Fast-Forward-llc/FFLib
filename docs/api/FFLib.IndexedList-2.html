Class IndexedList<TKey, TValue>

A list of items indexed by a key. Items with duplicate keys are stored in a list by that key. Key Capacity is the number of keys the list is preallocated to hold, it will expand as needed. Item Capacity is the number of items per Key (i.e. duplicate keys) the per Key Lists are preallocated to hold, they expand as needed.
